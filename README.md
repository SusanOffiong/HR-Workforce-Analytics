# 👥 HR Workforce Analytics

> **Data Analytics Case Study | Human Resources**

---

# Project Overview

This project explores employee data to help organisations better understand workforce trends and support informed human resource decisions.

Using SQL, I analysed employee information to identify workforce patterns, answer key HR business questions, and present insights that can support employee management, workforce planning, and organisational decision-making.

The project demonstrates how data can be used to move beyond reporting and provide meaningful insights for HR professionals.

---

# The Challenge

Human Resources teams collect large amounts of employee data, but without proper analysis it can be difficult to identify trends and make informed workforce decisions.

The objective of this project was to analyse employee data and provide insights that support:

- Workforce planning.
- Employee management.
- Organisational reporting.
- Better HR decision-making.

---

# My Approach

To complete this project, I:

- Queried employee data using SQL.
- Cleaned and organised the dataset.
- Analysed workforce trends.
- Answered key HR business questions.
- Interpreted the findings.
- Presented practical business insights.

---

# The Solution

The analysis provides HR stakeholders with valuable insights into workforce data, helping them better understand employee trends and make evidence-based decisions.

The project demonstrates how SQL can be used to answer real business questions using employee data.

---

# Business Impact

The analysis supports organisations by helping them:

- Better understand workforce trends.
- Improve workforce planning.
- Support HR reporting.
- Make informed people management decisions.
- Identify patterns within employee data.
